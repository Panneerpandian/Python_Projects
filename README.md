This project is an interactive supply chain analytics dashboard built with Streamlit.

## Features
- Fill Rate and OTIF service KPIs
- Carrier performance visualization
- Days of Supply (DoS) inventory tracking
- Stockout risk forecasting with lead time comparison
- Risk summary with severity indicators

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run dashboard.py
